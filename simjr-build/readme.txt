This is the Sim Jr build project. It is an Eclipse project. It includes and
Eclipse PDE target definition (default.target) as well as all of the 3rd
party plugins required by Sim Jr.

See https://socialtext.soartech.com/irad/index.cgi?sim_jr_setup_instructions
for how to run and build Sim Jr.

To do a full distribution, have ANT 1.7.1 or newer installed and run:

> build

this will produce simjr-x.x.x.zip in this directory.
